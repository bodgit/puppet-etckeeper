Puppet Class: etckeeper

Inherits:
::etckeeper::params

Overview

Installs etckeeper.

Examples:

Declaring the class

include ::etckeeper

Choosing a specific VCS instead of the default

class { '::etckeeper':
  vcs => 'bzr',
}

Forcing the user name & email for commits

class { '::etckeeper':
  vcs            => 'git',
  vcs_user_name  => 'Alice',
  vcs_user_email => 'alice@example.com',
}
